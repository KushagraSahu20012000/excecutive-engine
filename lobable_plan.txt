Executive Engine

A mobile-first installable web app (PWA, installable to Android home screen) for people with ADHD. Bento-system layout, Swiss typography, minimal foundation, subtle glass + motion accents. Cloud sync via login.

Pages (bottom tab nav, mobile-first)





Today — daily repeating tasks with the "pencil tick" checkbox



Goals — long-term goals with countdown timer + daily checklist + what works / what doesn't



Deadlines — one-off deadlines with date+time and pass/fail mark



Stats — weekly progress line chart + deadlines pass/fail ratio



Auth — /login (email + password, Google sign-in via Lovable broker)

Feature detail

Today (daily tasks)





Add / edit / remove repeating tasks



Custom hand-drawn checkbox: SVG checkmark animates beyond box bounds with a pencil-stroke easing (Motion / framer-motion)



Toggle "Include Saturday" and "Include Sunday" (default off → 5-day week, so 5 tasks × 5 days = 25)



Daily checks reset at local midnight (stored as date-stamped completion rows)



Week summary card on this page:





completed / total for the active week



Percentage + dynamic encouragement copy:







0% "Great going"



≥25% "Possible victory"



≥50% "You did great"



≥75% "Almost at peak"



100% "You have achieved greatness"

Goals





Title, description, deadline (date+time) → live ticking countdown (days / hrs / mins / secs)



"Daily actions" sub-checklist (things to do every day toward this goal)



Two side-by-side bento panels: What works / What doesn't (add / remove entries)

Deadlines





Fields: title, optional description, date+time, status (Pending → Pass / Fail toggle once due)



Sorted by soonest; overdue without mark shown as "Awaiting outcome"

Stats





Line chart: weekly completion % across the last 8 weeks (Recharts)



Donut/ratio: deadlines Passed vs Failed vs Pending



Streak count (consecutive days at ≥50%)

Visual system





Foundation: Minimal off-white background oklch(0.98 0.005 80) / deep ink oklch(0.18 0.02 260) for dark



Bento: rounded-2xl cards on a 12-col responsive grid, each with strict hierarchy



Accent: subtle backdrop-blur glass on the active tab bar + countdown card; restrained motion (spring on tick, fade-slide on route change)



Typography (Swiss): display = Space Grotesk, body = Inter — tight tracking, large numeric weights for stats/countdown



Single accent color: electric indigo oklch(0.58 0.22 265) for actions/progress



All tokens defined in src/styles.css (oklch); no hex in components

Tech / architecture





TanStack Start routes:





/login, /signup, /reset-password (public)



_authenticated/index → Today



_authenticated/goals, _authenticated/goals/$goalId



_authenticated/deadlines



_authenticated/stats



Lovable Cloud enabled. Tables (RLS = owner only via auth.uid()):





profiles (id, display_name)



settings (user_id PK, include_saturday bool, include_sunday bool)



tasks (id, user_id, title, position, created_at, archived_at)



task_completions (id, user_id, task_id, date) — unique(user_id, task_id, date)



goals (id, user_id, title, description, deadline_at, created_at)



goal_daily_actions (id, goal_id, user_id, title)



goal_notes (id, goal_id, user_id, kind enum 'works'|'doesnt', body)



deadlines (id, user_id, title, description, due_at, outcome enum 'pending'|'pass'|'fail')



Each table: GRANTs for authenticated + service_role, RLS policies scoped to auth.uid(), plus a trigger to auto-create profiles + settings row on signup



Data read via createServerFn + requireSupabaseAuth; loaders use ensureQueryData + useSuspenseQuery



Mutations via useMutation → invalidate queries → optimistic check toggle for snappy feel



Recharts for line/donut; date-fns for week math; framer-motion for the pencil-tick animation



Web App Manifest only (no service worker) — installable to Android home screen as the user requested; no PWA caching/SW per template guidance
